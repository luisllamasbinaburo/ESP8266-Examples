## Designación de los pines
```cpp
digitalWrite(D5, LOW);  // D5, que es igual a GPIO14

digitalWrite(14, LOW);  // GPIO14 directamente
```


## Salidas y entradas digitales (GPIO)
```cpp

```

```cpp
digitalWrite(pin, output)
```

```cpp
digitalRead(pin)
```


## Salidas analógicas (PWM)
```cpp

```

```cpp

```

```cpp

```

